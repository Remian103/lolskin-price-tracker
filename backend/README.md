# Installation
## Install Poetry
https://python-poetry.org/docs/#installation

## Setup environment
```bash
poetry install
```

## Activate environment
```bash
poetry shell
```
which will open a new shell with environment on

or


- Linux
```bash
source {path_to_venv}/bin/activate
```
 - Windows
```bash
source {path_to_venv}\Scripts\activate.bat
```
